# 📊 Customer Purchase Analysis Dashboard (Streamlit App)

This project is a **Streamlit-based data science dashboard** that performs an in-depth analysis of customer purchase behavior using visualizations, machine learning, and statistical metrics.

## 🔍 Features

- 📦 Product-wise classification of sales and engagement
- 🎯 Customer engagement analysis
- 🏷️ Coupon usage trends by year and product type
- 🧼 Automated data cleaning & preprocessing
- 📈 Month-wise sales trends and heatmaps
- 🔧 Feature Engineering (like PAT Income)
- 🤖 Linear Regression model with accuracy, F1 Score & residual analysis
- 📉 Error analysis graph post model prediction
- 🌐 Deployed with **Streamlit**
